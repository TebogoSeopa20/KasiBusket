## 🚀 Deployment Instructions

### Prerequisites

- Huawei Cloud account
- Node.js 18+ and npm
- Supabase project (optional)

### Step 1: Huawei Cloud Setup

```bash
# Configure Huawei Cloud credentials
export HUAWEI_ACCESS_KEY="your_access_key"
export HUAWEI_SECRET_KEY="your_secret_key"
export HUAWEI_REGION="af-south-1"
export HUAWEI_PROJECT_ID="your_project_id"
```

### Step 2: Enable Huawei Cloud Services

1. **OBS**: Create bucket `spaza-eats-storage`
2. **ModelArts**: Deploy AI models for recommendations
3. **Cloud Eye**: Set up monitoring dashboards
4. **API Gateway**: Configure rate limiting rules
5. **CloudTable**: Create tables for real-time sync
6. **Blockchain**: Initialize BCS network
7. **Maps AI**: Enable routing API

### Step 3: Application Deployment

```bash
# Install dependencies
npm install

# Build application
npm run build

# Deploy to Huawei Cloud ECS
# (Use Huawei Cloud CLI or console)
```

### Step 4: Configure Environment

Update `/services/HuaweiCloudService.ts` with your credentials:

```typescript
this.config = {
  region: 'af-south-1',
  projectId: 'YOUR_PROJECT_ID',
  accessKeyId: 'YOUR_ACCESS_KEY',
  secretAccessKey: 'YOUR_SECRET_KEY',
  obsEndpoint: 'https://obs.af-south-1.myhuaweicloud.com',
  modelArtsEndpoint: 'https://modelarts.af-south-1.myhuaweicloud.com',
  apiGatewayEndpoint: 'https://apig.af-south-1.myhuaweicloud.com'
};
```
