
  # kgodi's System spazaShop

  This is a code bundle for kgodi's System spazaShop. The original project is available at https://www.figma.com/design/QfvavGh9num7kWCaazKaZr/Peter-Adjao-E-commerce-Product-Catalogue-design.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  
  ### Customising branding & background

  * To change the loading screen or app background you can drop image files into `public/assets` (create the folder if it doesn't exist). By default the app looks for:
    * `/assets/landing-bg.jpg` – full‑screen background shown on most pages
    * `/assets/logo-spinner.png` – small logo that spins while loading
  * If you prefer to use your own CSS/URL just update `src/App.tsx` or the new `LoadingScreen` component.
